# 1977 — CHIP-8 and Personal Computing

## Computing Comes Home
1977 was the year personal computing became real. Three machines launched 
that year — the Apple II, the TRS-80, and the Commodore PET — brought 
computers into homes for the first time. That same year Joseph Weisbecker 
at RCA created CHIP-8, a simple virtual machine that made programming these 
hobbyist computers accessible to everyone.

## What Was CHIP-8?
CHIP-8 was a programming language and interpreter designed to run on early 
hobbyist computers. Instead of writing complex machine code specific to one 
computer, programmers could write CHIP-8 programs that ran on any machine 
with a CHIP-8 interpreter. It was one of the earliest examples of write once 
run anywhere software — the same idea behind Java twenty years later.

## The Hobbyist Era
In 1977 computers came as kits. Enthusiasts would order parts, solder them 
together, and program them directly in binary machine code. CHIP-8 was 
revolutionary because it gave these hobbyists a simple, friendly way to 
write programs — especially games. Pong, Space Invaders, and Tetris were 
all written for CHIP-8.

## The Lasting Legacy
CHIP-8 never went away. In the 1990s it was revived for the HP48 calculator. 
Today it is the first emulator every emulator developer builds as a learning 
project — including this one. Its simplicity makes it the perfect bridge 
between the theoretical machines of the 1940s and the complex systems of 
the modern era.
