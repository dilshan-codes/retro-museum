# 1948 — Manchester Baby

## The Day Computing Became Real
On June 21, 1948, at the University of Manchester, a machine called the 
Small-Scale Experimental Machine ran the world's first stored program. 
Before this day, computers existed — but they had to be physically rewired 
for every new task. The Baby changed that forever.

## The Team
Freddie Williams and Tom Kilburn built the Baby to test a new memory 
technology called the Williams Tube — a cathode ray tube that stored binary 
data as dots of electric charge on a glass screen. Engineer Geoff Tootill 
assisted in the construction.

## The First Program
Tom Kilburn wrote the first program. It found the highest factor of a given 
number by trying every possible divisor from the largest downward. On June 
21 it ran successfully, taking 52 minutes and 3.5 million operations to find 
the answer. It was the first time in history a computer had solved a problem 
using a program stored in its own memory.

## What Made It Revolutionary
The stored program concept means the program lives in memory alongside the 
data it works on. You change what the computer does simply by loading a 
different program — no rewiring required. This is why you can use the same 
phone to send a message, play a game, and take a photo. The Baby made that 
world possible in 1948.
